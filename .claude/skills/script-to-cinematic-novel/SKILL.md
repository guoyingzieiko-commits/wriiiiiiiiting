# 剧本影视化小说改写（Script to Cinematic Novel）

把用户提供的中文剧本原文，改写为**小说细腻语言 + 镜头语言**融合的叙事正文。改写的目标是：读者读文字时，脑中能自动"放电影"——画面可拍摄、节奏有呼吸、情绪有张力。

## 第零步：先读风格学习案例库（最高优先级）

**开始任何改写之前，必须先读取 [resources/example-pairs.md](resources/example-pairs.md)**，从其中的"剧本原文 → 优化后正文"对照案例中学习写法。

从案例中学习以下维度，并让输出贴近案例的语言质感：

- **扩写密度**：一行剧本动作对应多少叙事文字，案例怎么扩你就按相近密度扩。
- **句子节奏**：长短句的搭配比例、何处用短句急收、何处用长句铺陈。
- **情绪表达方式**：情绪如何外化到动作和身体细节，外化到什么程度。
- **镜头语言嵌入方式**：景别切换的时机、隐性运镜的写法、术语词出现的频率。
- **动作细节颗粒度**：手部、视线、衣料、器物等细节写到多细。
- **对白保留和微调方式**：核心台词如何原样保留、节奏如何拆分、停顿如何插入。
- **分场景风格**：古装、现代、悬疑、暧昧等不同类型的用词和笔触差异——优先匹配与当前剧本**同类型**的案例；若无同类型案例，参考整体语言质感。

**使用案例的边界（必须遵守）：**

1. 案例只用于学习**写法和质感**，**严禁**复制或挪用案例中的具体剧情、人物、场景设定和句子。
2. 输出内容必须完全来自用户输入的剧本：保留其剧情、人物关系、事件顺序和关键信息，不擅自新增重大剧情。
3. 不输出任何分析和说明，只输出优化后的正文。
4. 若案例库为空或只有占位示例，则按本文件其余章节的规则独立完成改写。

## 核心原则（必须严格遵守）

### 一、忠于原作 —— 改写不是改编

1. **完整保留**原剧情走向、人物关系、事件顺序、关键信息点和核心台词。
2. **禁止**擅自新增重大剧情、新增角色、新增关键道具或伏笔。
3. **禁止**改变人物动机、立场和性格底色。
4. **禁止**改变任何冲突的结果（谁赢谁输、谁妥协、谁离开，必须与原剧本一致）。
5. 对白可以微调节奏（拆句、加停顿、调整语序），但**不能改变原意**，核心台词原句保留。
6. 原剧本中省略或含糊的部分，只允许做"镜头级"的合理补足（环境、动作、神态），不允许补足成新情节。

### 二、扩写方式 —— 从"剧本指令"到"可拍摄的叙事"

把剧本中的简单动作指令，扩写成有画面感、可拍摄的连续叙事：

- 剧本写"她转身离开"→ 扩写出转身前的迟疑、衣料摩擦声、脚步落点、背影在光线中的变化。
- 每个动作要回答三个问题：**镜头在哪里？观众看到什么？听到什么？**
- 扩写颗粒度：一行剧本动作 ≈ 一到三段叙事；一场戏 ≈ 一个有起伏的完整段落组。
- 所有描写必须**可拍摄**：不写"她感到世界崩塌了"这类无法入镜的抽象句，写"她扶住门框，指节一点点收紧，泛出白色"。

### 三、镜头语言 —— 自然织入，不堆术语

在叙事中自然融入镜头语言，让文字自带运镜感。可用的手法包括（不限于）：

| 镜头手法 | 文字化用法示例 |
|---------|--------------|
| 远景 / 大全景 | 开场定环境、定氛围："暮色压下来，整座宫城只剩檐角一线金。" |
| 中景 | 交代人物关系和站位："两人隔着一张棋盘对坐，谁也没先落子。" |
| 特写 | 放大关键细节："茶盏沿口，一道极细的裂纹。" |
| 镜头推进 | 由远及近逐层聚焦，制造压迫或亲密感 |
| 反打 | 对话双方视角交替，写"他看到的她"与"她看到的他" |
| 空镜 | 情绪留白：风吹帘动、烛火一跳、雪落无声 |
| 声音先入 | 画面未到声先到："脚步声停在门外。很久，门才开。" |
| 拉远 / 升格 | 收尾抽离，余味："镜头般地退远，长街尽头只剩一盏灯。" |

使用规则：

- 镜头感通过**描写顺序和焦点变化**体现，优先"隐性运镜"；术语词（特写、推近等）可少量点缀，**不要每段都标注机位**。
- 场景开头优先建立空间感（远景/全景），关键情绪节点切特写，场与场之间可用空镜或声音先入转场。
- 一场戏内焦点要有层次变化，避免全程同一景别。

### 四、细节强化清单

每场戏检查是否调用了以下维度（不必全用，按戏的需要取舍）：

- **光影**：光源、明暗交界、影子的位置和移动。
- **环境**：天气、温度、气味、背景声（更声、雨声、电流声）。
- **动作细节**：手部小动作、衣料、器物的触碰与声响。
- **停顿与呼吸**：对白之间的沉默时长、呼吸的轻重变化。
- **视线**：谁先看谁、视线落点、移开视线的时机。
- **人物距离**：物理距离的拉近与退开，即是关系的拉近与退开。
- **情绪张力**：用"将发未发"的状态制造悬置感。

### 五、情绪表达 —— 外化，不解释

- 情绪**通过动作和身体细节表现**：手指、肩线、喉结、呼吸、脚步、视线。
- **禁止**大段心理独白和心理解释。不写"她心里又恨又怕，想起多年前……"，写"她垂下眼，拇指反复摩挲着袖口那道旧补丁"。
- 偶尔可用一句极短的内心声音点睛（一句以内），但不可成段。

## 类型适配

根据剧本类型自动调整笔触，类型由剧本内容判断，不需要用户声明：

- **古装权谋**：用词典雅克制，多写礼制、器物、站位与称谓中的权力关系；张力藏在"没说出口的话"里。
- **女频短剧**：节奏快、钩子密，情绪浓度高但落点准；重视脸部特写、对视、肢体接触的瞬间放大。
- **悬疑探案**：信息释放有控制，环境描写带不安感；特写多用于线索物件，善用声音先入与视线引导。
- **暧昧拉扯**：核心在距离与呼吸——靠近又退开、将触未触；多用慢节奏特写和留白，台词少而轻。
- **末世科幻**：质感冷峻，强调材质、机械声、光源稀缺；用大远景写废土的空旷，用特写写幸存者的生理细节。
- 其他类型：参照以上原则，匹配该类型的影视语言惯例。

## 输出规范

1. **默认只输出优化后的正文**，不输出任何分析、说明、改写思路或前后言。
2. 不保留剧本格式标记（"场景一""△""VO""人名：台词"等），全部转化为小说叙事体；对白用引号自然嵌入叙述。
3. 原剧本的分场可转化为正文的自然分段或空行分节，保持事件顺序不变。
4. 篇幅一般为原剧本的 2–4 倍；若用户指定字数或风格要求，以用户指定为准。
5. 若用户明确要求"附带说明/标注镜头"，才在正文之外补充，否则一律省略。
6. 若用户提供的剧本过长，按场次顺序完整改写，不得跳场、缩略合并场次。

## 工作流程

1. **读取 resources/example-pairs.md**，学习案例的写法和语言质感，优先匹配与当前剧本同类型的案例。
2. 通读剧本，识别：类型、人物关系、事件链、核心台词、冲突结果。
3. 按场次逐场改写：定景别 → 织入光影环境 → 扩写动作 → 外化情绪 → 微调对白节奏，全程贴近案例的扩写密度和句子节奏。
4. 自查一遍核心原则（剧情无新增、动机无改变、结果无改变、核心台词在场；未挪用案例中的剧情、人物和句子）。
5. 按输出规范，只输出最终正文。
