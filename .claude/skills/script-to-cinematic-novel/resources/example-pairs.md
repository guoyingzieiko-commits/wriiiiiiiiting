# 风格学习案例库（example-pairs.md）

本文件存放"剧本原文 → 优化后正文"的对照案例，供 skill 在改写时学习风格。

## 使用说明（给维护者）

- 每个案例用一个 `## 案例 N` 区块，按下方模板填写。
- **类型标签**必填，用于按场景匹配风格（如：古装权谋 / 现代都市 / 女频短剧 / 悬疑探案 / 暧昧拉扯 / 末世科幻）。
- 案例数量不限，越多越好；同一类型可放多个案例，覆盖不同情绪强度的戏（对峙、告白、追凶、独处等）。
- 案例只用于**学习写法**（密度、节奏、镜头嵌入方式），改写输出时绝不照搬案例中的剧情、人物和句子。
- 不再需要的案例直接删除区块即可。

## 案例模板（复制此区块新增案例）

```markdown
## 案例 N：[一句话概括这场戏，如"书房对峙"]

- 情绪基调：[如：压抑、克制的对峙 / 慢热升温 / 紧张追逐]
- 风格要点（可选）：[这条案例特别值得学习的地方，如"对白间停顿的写法""特写切入时机"]

[粘贴剧本原文]

### 优化后正文

[粘贴修改后的可用内容]
```

## 案例 1：（占位示例）书房对峙

- 类型：古装权谋
- 情绪基调：压抑、克制的对峙
- 风格要点：用烛火和墨滴写停顿；情绪全部外化到手部动作

### 剧本原文

> 夜。书房。
> 沈相端坐。林晚入，跪。
> 林晚：大人，账册找到了。
> 沈相：（停顿）烧了。
> 林晚愕然，欲言又止，退下。

### 优化后正文

> 夜漏过半，书房里只点了一盏灯。灯焰极稳，把沈相的影子钉在身后整面书架上，纹丝不动。
>
> 门外先传来脚步声，很轻，到门槛处停了一瞬。林晚推门进来，带进一缕夜里的凉气，烛火晃了晃。她屈膝跪下，双手把那本账册举过头顶，指尖还带着夜露的潮意。
>
> "大人，账册找到了。"
>
> 沈相没有看她。他执笔的手悬在纸面上方，一滴墨慢慢坠下去，洇开一团黑。许久——
>
> "烧了。"
>
> 林晚抬起头。烛光照见她眼里那一点骤然睁大的茫然，嘴唇动了动，终究什么也没说。她低下头，把账册收回怀里贴紧，起身，一步一步退出门去。
>
> 门合上。灯焰又稳了。沈相落下笔，那个字写得很重，力透纸背。
